## Key Implementation Details

### authMiddleware.js
- Reads token from `Authorization: Bearer` header AND `?token=` query string
- Query token fallback needed for `window.open()` PDF downloads (can't set headers)
```js
if (!token && req.query.token) token = req.query.token;
```

### Application.js — activityLog schema
```js
activityLog: [{
  action: enum["status_changed","fields_edited","created","submitted","payment_completed"],
  by: String,       // admin email
  fromValue: String,
  toValue: String,
  note: String,
  at: Date,
}]
```

### Application.js — indexes
```js
{ status: 1 }
{ createdAt: -1 }
{ userId: 1 }
{ applicationNumber: 1 }       // for exact lookups in PDF — keep alongside text index
{ paymentStatus: 1 }
{ status: 1, createdAt: -1 }   // compound
// Text index:
{ fullName: "text", applicationNumber: "text", email: "text", contactNo: "text" }
// weights: applicationNumber:10, contactNo:5, email:5, fullName:1
// $text search: query.$text = { $search: search }  — NO forced phrase quotes
```

### AdmissionSettings.js
- Singleton: `AdmissionSettings.getSettings()` always returns/creates one doc
- `isAccepting` virtual: `isOpen && now >= openDate && now <= closeDate`

### applicationController.js — session-scoped duplicate check
```js
// settings already fetched above for portal-open check — no extra DB call
Application.findOne({ userId: req.user.id, session: settings.session })
// Returns 409 DUPLICATE_APPLICATION with session name in message
```

### adminController.js — changePassword
- Finds admin with `User.findById(req.user.id).select("+password")`
- Verifies current password with `admin.comparePassword(currentPassword)`
- Sets `admin.password = newPassword` — hashed automatically by `pre-save` hook in User.js
- Returns 401 if current password is wrong, 400 if same as new password

### admin.js — CRITICAL route ordering
```js
// ✅ CORRECT — specific routes before parameterised
router.get("/applications/:id/activity", protect, adminOnly, getActivityLog);
router.get("/applications/:id", protect, adminOnly, getApplicationById);
// ❌ WRONG — if /:id comes first, Express treats "activity" as the :id value
```

### api.js — PDF URLs must be ABSOLUTE
```js
// ✅ Correct — hits Express on port 5000
const base = import.meta.env.VITE_API_URL || "http://localhost:5000";
return `${base}/api/pdf/application-summary/${id}?token=${token}`;
// ❌ Wrong — resolves to Vite dev server on port 5173
return `/api/pdf/application-summary/${id}?token=${token}`;
```

### api.js — key adminAPI methods
```js
getAllApplications(params), getApplicationById(id), getActivityLog(id),
updateApplicationStatus(id, status), updateApplication(id, data),
createAdmin(data), adminLogin(data),
getSettings(), updateSettings(data),
changePassword(data)   // PUT /admin/change-password
```

### api.js — settingsAPI
```js
getSettings: () => api.get("/admin/settings"),   // public — no auth needed
updateSettings: (data) => api.put("/admin/settings", data),
```

### AdminDashboard.jsx — bulk selection
- `eligibleIds` = only `submitted` / `under_review` apps can be checked
- Checkboxes use inline style `{ accentColor: "#6366F1" }` — NOT Tailwind classes
  (Tailwind checkbox styling requires `@tailwindcss/forms` plugin which is NOT installed)
- Bulk action bar appears when `selectedIds.length > 0`
- Re-approve button (green circle-check) shown on `rejected` rows only

## Critical Rules (do not break)

| Rule | Detail |
|------|--------|
| Documents stored as `.filename` only | NOT `.path` — served via `/uploads/` static route in server.js |
| server.js must have static middleware | `app.use('/uploads', express.static(path.join(__dirname, 'uploads')))` |
| Do NOT add extra index on `applicationNumber` | Already has `unique:true` + covered by text index |
| No custom `keyGenerator` in rate limiters | Causes `ERR_ERL_KEY_GEN_IPV6` in express-rate-limit v7+ |
| Never mix `documents` + `documents.fieldName` in same `$set` | MongoDB throws conflict error |
| Activity route MUST come before `/:id` in admin.js | Express matches "activity" as :id param otherwise |
| PDF download URLs must be absolute | Relative URLs resolve to Vite port 5173, not Express port 5000 |
| Checkbox styling: use inline style only | `{ accentColor: "#6366F1" }` — Tailwind needs `@tailwindcss/forms` plugin |
| $text search: no forced quotes | `{ $search: search }` not `{ $search: \`\"${search}\"\` }` |
